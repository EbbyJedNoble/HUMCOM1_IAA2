How to open the door

Materials needed:

    • Door key

Steps:

  1. Locate and approach the door.
  2. Inspect the door if it is locked or not
    • If it is locked, equip the door key.
    • Insert the door key into the door lock and rotate it either clockwise or anticlockwise depending on the location of the door lock on the door in order to unlock the door.
  3. Inspect what kind of door handle it is.
    • If it is a door knob, grab the handle and rotate it.
    • If it is a lever handle, grab the handle and pull it down.
    • If it is a pull handle, just grab onto it.
  4. Test to see if the door can be opened by pulling or pushing.
    • If the door can't be pulled then do the reverse by pushing, and vice versa if it can't be pushed.
  5. Apply the correct type of force and the door is opened.
